ER-диаграмма базы данных KodShopWeb

Для сдачи: откройте этот файл в браузере → Печать → «Сохранить как PDF» → Docs/ER-Diagram.pdf

AppUser

  • Id (PK)
  • FullName
  • Login
  • Password
  • Role
  • Email

Category

  • Id (PK)
  • Name

Manufacturer

  • Id (PK)
  • Name

Supplier

  • Id (PK)
  • Name

Product

  • Id (PK)
  • Article
  • Name, Description
  • Price, Unit
  • QuantityOnStock
  • Discount, ImagePath
  • CategoryId (FK)
  • ManufacturerId (FK)
  • SupplierId (FK)

PickupPoint

  • Id (PK)
  • Address

Order

  • Id (PK)
  • Article
  • Status
  • OrderDate
  • DeliveryDate
  • PickupCode
  • PickupPointId (FK)
  • ClientId (FK → AppUser)

OrderItem

  • Id (PK)
  • OrderId (FK)
  • ProductId (FK)
  • Quantity

Связи

  • Category 1 — N Product
  • Manufacturer 1 — N Product
  • Supplier 1 — N Product
  • PickupPoint 1 — N Order
  • AppUser (Client) 1 — N Order (необязательная)
  • Order 1 — N OrderItem
  • Product 1 — N OrderItem
